# 面向机器学习的工艺路线安全前置评估框架（论文写作包 v1）

## 1. 定位声明（可直接放引言）
本研究提出一种面向机器学习的工艺路线安全前置评估框架（ML-ready pre-assessment framework）。该框架不依赖监督学习训练，而是通过可解释规则引擎实现对反应物危害、反应条件、控制措施和分阶段操作风险的结构化量化，输出风险等级、风险驱动因素与缓解建议。框架的核心目标是：
- 在无大规模标注数据条件下提供可复现、可审计的风险评估基线。
- 生成高质量结构化特征，为后续监督学习模型训练提供输入。

## 2. 贡献点（可直接放贡献列表）
1. 提出路线级安全前置评估方法，将“化学危害 + 工艺条件 + 控制能力 + 分阶段风险”统一建模。
2. 构建远端危害数据查询与本地缓存机制，实现按需拉取与可追溯证据链。
3. 引入剂量/当量加权聚合与分阶段热图，使评估结果更接近真实工艺操作语境。
4. 在不训练ML模型的前提下，提供ML-ready特征输出与可解释决策基础。

## 3. 系统方法描述（方法章节模板）
### 3.1 输入层
- 反应物清单：支持名称/CAS/InChIKey。
- 反应条件：最高温度、最高压力、放热强度、放气潜势、失控潜势。
- 控制层信息：工程控制等级、监测覆盖、PPE依赖程度。
- 分阶段风险：投料、反应、淬灭、分离四阶段风险分级。
- 规模信息：实验室/中试/生产。

### 3.2 数据层
- 远端数据源：公开化学危害数据库接口（按需查询）。
- 本地缓存：TTL策略缓存查询结果，提升重复查询效率与可用性。
- 证据记录：每个反应物保留危害摘要与来源信息。

### 3.3 评估层
- 风险特征提取：危险标签计数、CMR标志、可燃性标志等。
- 路线聚合：对反应物危害按当量/投料量进行加权合成。
- 规则评分：综合化学危害、工艺条件、控制能力与阶段风险生成安全评分。
- 结果输出：风险等级（Low/Moderate/High/Critical）、置信度、驱动因子、缓解建议。

### 3.4 输出层
- 总体风险评分（0-100）
- 风险等级
- 主要风险驱动项
- 缓解建议
- 分阶段热图
- 反应物证据卡

## 4. 评价设计（当前阶段无需ML训练）
本阶段不报告“机器学习准确率”，而报告“前置评估框架有效性指标”：

1. 一致性指标（与专家）
- Overall Agreement
- Cohen's Kappa

2. 高风险识别能力
- High/Critical Recall
- High/Critical Precision

3. 排序能力（可选）
- AUROC
- PR-AUC（类别不均衡时优先）

4. 可解释性与可追溯性
- 每个预测样本是否输出驱动因子与证据链（覆盖率）
- 人工审阅可复核比例

## 5. 数据标注协议（最小可行）
1. 每条路线至少2名评估者独立打标。
2. 标签分四级：Low/Moderate/High/Critical。
3. 标注冲突由第三方仲裁。
4. 保存一致性元数据：标注者ID、时间戳、冲突类型、仲裁结论。

## 6. 论文文本模板（可直接替换）
### 6.1 摘要模板
本文提出一种面向机器学习的工艺路线安全前置评估框架。该框架通过整合反应物危害信息、工艺操作条件、工程控制措施与分阶段风险特征，实现路线级风险量化与可解释输出。系统采用远端危害数据按需查询与本地缓存机制，并引入当量/投料量加权聚合策略及阶段热图，以提升评估的操作语境一致性。与直接依赖监督学习的方法不同，本研究聚焦于构建可复现、可审计的前置评估层，为后续机器学习模型提供结构化高质量输入。实验结果表明，该框架能够稳定输出风险等级、关键驱动因子与缓解建议，具备良好的工程可落地性与扩展性。

### 6.2 方法描述模板
我们将工艺路线安全评估拆分为四类特征域：化学危害域、工艺条件域、控制能力域和分阶段操作域。系统首先对反应物进行远端危害检索并构建本地证据缓存，随后基于当量/投料量进行路线级危害聚合，最后采用可解释规则引擎生成风险评分与风险等级。该方法在无监督学习训练条件下即可运行，并可无缝扩展为监督学习输入层。

### 6.3 结果描述模板（当前版本）
由于当前研究阶段尚未构建大规模标注训练集，本文不报告机器学习模型准确率。我们重点报告前置评估框架在一致性、可解释性和可追溯性方面的表现。结果显示，系统可稳定生成路线级风险评分，并提供覆盖化学危害、工艺条件与阶段操作的风险证据，能够支持后续机器学习建模的数据准备与特征工程。

### 6.4 局限性与未来工作模板
本研究尚未完成监督学习模型训练与外部多中心数据验证，当前结果主要体现前置评估框架的工程可行性。未来将基于本框架生成的结构化样本开展监督学习建模与校准研究，并通过跨实验室数据验证模型泛化能力。
